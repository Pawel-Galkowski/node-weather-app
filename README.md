# Weather App

Application which are showing temperature in cities.

## Technology stack
    - Node.JS
    - JavaScript
    - HTML
    - CSS

## Installation
To install appliaction you need to have a npm.

At the start you need to type in manager

```npm
npm install
```

## Usage
To use application you need to type

```npm
npm start
```
